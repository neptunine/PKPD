To get up and running quickly, simply go to Window>Journal>Journal Manager. Journal Manager will check your
projec to see if the required components for TextMeshPro are imported. If they are, Step 1 of the installer will be marked done.
If not, simply click Import TMPRo Essentials. Once imported, click Install Journal (Step 2). 

Once Journal is installed, running your game will show the demo achievements in the UI. This data can be edited in the
Journal Manager. For a complete guide, check the online documentation.
